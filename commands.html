Commands

Command Function Usage
choice Get random answer from selected choices /choice [option 1]/[option 2] ...
dice Get random result from throwing a dice /dice
coin Get random result from flipping a coin /coin
answer Get random answer of a question from selected choices /answer [question] [choice 1]/[choice 2] ...
8ball Ask 8ball about your condition /8ball [question]
say Make Tycha say something /say [sentences]
randint get random integer number from given range /randint [begin] [end]
randfloat get random float number from given range /randfloat [begin] [end]
